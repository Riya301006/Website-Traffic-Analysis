# Website-Traffic-Analysis

## Project Overview
This project analyzes website traffic data to identify top traffic sources,
user engagement patterns, conversion rates, and business insights
using Python and data visualization tools.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Dataset
- Source: Google Analytics Export
- File: data-export (1).csv
- Contains: Sessions, Users, Engaged Sessions, 
  Engagement Rate, Events per Session by Traffic Source

## Project Steps
1. Data Loading & Exploration
2. Data Cleaning & Preparation
3. Traffic Source Analysis
4. User Engagement Metrics Analysis
5. Conversion Rate Tracking
6. Dashboard Creation

## Key Findings
- Organic Social drives the highest number of sessions
- Direct traffic shows strong user engagement
- Email has low volume but consistent engagement
- Referral traffic has growth potential
- Organic Search brings quality engaged users

## Dashboard
Interactive dashboard created using Plotly with 6 charts:
- Sessions by Traffic Source
- Users by Traffic Source
- Engagement Rate by Source (Donut Chart)
- Engaged Sessions by Source (Funnel Chart)
- Conversion Rate by Source
- Sessions vs Conversion Rate

## How to Run
1. Clone this repository
2. Open Jupyter Notebook
3. Run all cells from top to bottom
4. Dashboard will appear at the bottom
